02. Fizz Buzz
This is a popular challenge where the user provides a series of numbers (F B N). The game
starts counting from 1 up to N. Along the way if one of the numbers is divisible by F you write F. If a number is divisible by B you write B. In the event the number is divisible by both F AND B you write FB. When the number is not divisible by either F or B you write the number.

Example: 2 3 9
1 F B F 5 FB 7 F B
